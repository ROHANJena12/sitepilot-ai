# Feature: Recommendations

## Responsibility

Owns actionable recommendation lists produced from an audit — prioritized fixes, effort/impact cues, and navigation into deeper finding detail.

## Boundaries

- **In scope:** Recommendation list UI, filtering/sorting UX, and action CTAs.
- **Out of scope:** Business impact narrative (`features/business-impact`), report shell (`features/report`), and domain entity models (`entities/audit`, `entities/report`).

## Composition

Typically composed inside `widgets/report-view` and dashboard follow-up modules.

## Status

Scaffold only. Implementation pending.
