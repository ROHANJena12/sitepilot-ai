# Feature: Landing

## Responsibility

Owns the marketing landing experience for SitePilot AI — value proposition messaging, primary CTA to start an audit, and supporting narrative sections that convert visitors into users.

## Boundaries

- **In scope:** Landing-specific sections, CTA flows, and marketing copy modules.
- **Out of scope:** Global navigation/footer (see `widgets/`), audit execution (see `features/audit`), and shared UI primitives (see `shared/ui`).

## Composition

Consumed by `app/page.tsx` via `widgets/hero` and related landing sections.

## Status

Scaffold only. Implementation pending.
