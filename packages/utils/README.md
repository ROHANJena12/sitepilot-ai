# packages/utils

**Responsibility:** Shared, framework-agnostic utility helpers.

## Scope

- Pure helpers (formatting, validation, URL/path utilities)
- Cross-app constants that are not domain types
- No UI, no I/O-heavy side effects

## Consumers

- `apps/web`
- `apps/api` (where language-compatible)
- Other packages

## Status

Scaffold only. No utilities are implemented yet.
